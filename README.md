# 🗣️ Angine de Poitrine Universal Translator

Un traducteur expérimental qui transforme du texte en langage **Angine de Poitrine™** — un dialecte synthétique, phonétique et chaotiquement déterministe.

👉 Essaye-le en ligne :
[https://zmotrin.github.io/adp-translator/](https://zmotrin.github.io/adp-translator/)

---

## ✨ Features

* 🔊 Génération phonétique basée sur **double metaphone**
* 🎲 Randomness **déterministe** (même input = même output)
* 🧠 Lexique interne stable
* 🔒 Support de **mots réservés** (expressions fixes)
* 🎛️ Système de **pulses** (timing audio / synthèse)
* 🧬 Mutation légère pour ajouter du “flavor”

## 📦 Installation (dev)

```bash
git clone https://github.com/ZmotriN/adp-translator.git
cd adp-translator
npm install
code .
```

### 📤 Export (site statique)


### 👀 Watch mode

Le mode watch est normalement actif par défaut.


## 🎭 Projet artistique

Ce projet s’inscrit dans une approche volontairement **absurde et expérimentale**, inspirée du **dadaïsme**.

Le but n’est pas la précision linguistique, mais la création d’un langage chaotique, sonore et expressif — quelque part entre bruit, poésie et erreur contrôlée.
